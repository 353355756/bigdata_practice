大数据实践项目 - nginx 日志分析可视化

## 功能说明

通过 hive 分析 nginx 日志，将分析结果通过 flask + echarts 进行可视化展示

## 数据收集分析过程

[数据收集分析过程](https://github.com/TurboWay/bigdata_practice/blob/master/process.md)


## 运行

运行 cd ironman; python app.py

打开 http://127.0.0.1:5000/
